# 后续开发计划 (Development Plan 2.0)

本计划基于已完成的基础闭环系统，旨在提升游戏的深度、视觉表现和内容丰富度。

## Phase 6: 视觉与表现力增强 (Visual & Juice)
目标：摆脱“简陋”的标签，让交互更具爽感。
- [x] **特效系统升级 (VFX Upgrade)**
    - [x] 实现通用的 `ParticleSystem` (粒子系统)，支持配置化。
    - [x] **渡劫特效**: 屏幕震动 + 雷电粒子 + 金光爆闪。
    - [x] **点击特效**: 鼠标点击时产生水波纹或灵气光点(及爱心)。
    - [x] **突破特效**: 升级时播放全身光环动画。
- [x] **动画增强 (Animation)**
    - [x] 为桌宠增加 `Idle` 变体动画 (如: 打瞌睡、伸懒腰、四处张望)，减少呆板感。
    - [x] 增加 `Drag` (拖拽) 时的悬空挣扎动画。
    - [x] 增加 `Interact` (抚摸/点击) 时的反馈动画 (爱心冒泡)。

## Phase 7: 内容扩充与平衡 (Content & Balance)
目标：增加游戏时长和策略深度。
- [ ] **Tier 3 (金丹期) 内容**
    - [ ] 新增材料: `千年魂木`, `深海寒铁`, `天火液` 等。
    - [ ] 新增丹药: `结金丹` (突破用), `神识丹` (大幅加 Mind 恢复).
    - [ ] 新增装备/法宝系统: 也许可以装备一个“法宝”在身边环绕飞行？
- [ ] **事件库扩充 (Event Pool)**
    - [ ] 编写至少 20+ 条随机事件，包含选择题 (risk vs reward)。
    - [ ] 实现“奇遇”链式任务 (连续触发的事件)。
- [ ] **成就系统 (Achievements)**
    - [ ] 记录生涯数据 (总点击、总APM、炼丹失败次数)。
    - [ ] 达成成就像获得徽章或特殊称号 (显示在名字旁)。

## Phase 8: 高级玩法 (Advanced Gameplay)
目标：利用桌面特性做更多有趣的事。
- [ ] **桌面互动 (Desktop Interaction)**
    - [ ] **窗口吸附**: 自动吸附到其他窗口边缘“观察”你在做什么。
    - [ ] **简易提醒**: 设置一个“番茄钟”炼丹，时间到了强力提醒。
- [ ] **放置挂机深度化**
    - [ ] **灵田系统**: 在桌面上开辟一小块(另外一个微型窗口)种草药，需要定时浇水。
    - [ ] **探索派遣**: 将宠物派遣出去(消失一段时间)，带回稀有战利品。

## 技术债务与优化 (Tech Debt & Optimization)
- [ ] **数据结构优化**: 将各类配置 (Items, Events, Dialogues) 拆分到更加规整的 JSON/YAML 目录中。
- [ ] **国际化 (i18n)**: 提取所有中文字符串，支持英文切换。
- [ ] **打包发布**: 配置 PyInstaller `spec` 文件，确保资源正确打包，制作 .dmg/.exe 安装包。

## 事件详情

### 0. 炼气期 (Level 0)

1.  **灵气复苏 (Common)**
    *   **描述**: 今日晨练时感觉神清气爽，似乎空气中的灵气浓度有所上升。
    *   **效果**: 修为 +50, 心魔 -2
2.  **误食野果 (Common)**
    *   **描述**: 你在路边发现一颗颜色鲜艳的果实，饥渴难耐下吞服了它。
    *   **效果**: (无直接效果)
3.  **兽冢拾荒 (Rare)**
    *   **条件**: 工作状态 (WORK)
    *   **描述**: 你误入一处荒废的兽冢，在白骨堆中发现了一颗泛着红光的残齿。
    *   **效果**: 获得 [狼牙 (part_wolf_tooth)] x1
4.  **神秘符箓 (Rare)**
    *   **条件**: 心魔上限 >= 20? (Data: mind_max: 20 -> 可能是误写，或是指当前mind < 20? 或是mind上限要求) *注: 数据显示 `mind_max: 20`*
    *   **描述**: 一位路过的云游道士看你骨骼惊奇，随手丢给你一张破旧符纸。
    *   **效果**: 获得 [旧符纸 (misc_talisman_old)] x1
5.  **洗髓机缘 (Unique)**
    *   **描述**: 你在深山瀑布下冲刷肉身时，意外发现瀑布后的一株幽蓝草药。
    *   **效果**: 获得 [洗髓草 (herb_marrow_0)] x1, 修为 +500

### 1. 筑基期 (Level 1)

1.  **地脉震动 (Common)**
    *   **描述**: 脚下的大地微微颤抖，似有地龙翻身，泄露出一丝精纯土灵气。
    *   **效果**: 修为 +200
2.  **坊市捡漏 (Common)**
    *   **条件**: 摸鱼状态 (IDLE)
    *   **描述**: 在坊市角落的摊位上，你发现一块不起眼的矿石竟是赤铜精母。
    *   **效果**: 灵石 -100, 获得 [赤铜 (ore_copper_red)] x1
3.  **寒潭奇遇 (Rare)**
    *   **条件**: 工作状态 (WORK)
    *   **描述**: 为了躲避仇家，你潜入寒潭深处，却意外发现寒气逼人的晶石。
    *   **效果**: 获得 [寒晶 (ore_cold_crystal)] x1, 体魄 +1
4.  **龙纹觉醒 (Unique)**
    *   **描述**: 你体内的灵力忽然自行运转，与手中的龙纹草产生共鸣，隐约听到了龙吟。
    *   **效果**: 获得 [龙纹草 (herb_dragon_1)] x1, 修为 +2000

### 2. 金丹期 (Level 2)

1.  **丹火试炼 (Common)**
    *   **条件**: 炼丹状态 (ALCHEMY)
    *   **描述**: 炉温突然升高，你急忙打出几道法诀稳住火候。
    *   **效果**: 修为 +500, 心魔 +1
2.  **天雷滚滚 (Rare)**
    *   **描述**: 晴空一声霹雳，一道落雷击中了你不远处的竹林！
    *   **效果**: 获得 [天雷竹 (wood_sky_thunder)] x1, 心魔 +5
3.  **古修洞府 (Rare)**
    *   **条件**: 工作状态 (WORK)
    *   **描述**: 你挖掘到了一座坍塌的洞府，门口散落着几面残破的阵旗。
    *   **效果**: 获得 [阵旗 (misc_array_flag)] x1, 灵石 +500
4.  **金丹异象 (Unique)**
    *   **描述**: 你的金丹表面浮现出九窍玲珑之孔，贪婪地吞噬着周围的星辰之力。
    *   **效果**: 获得 [星辰沙 (ore_star_sand)] x2, 修为 +5000

### 3. 元婴期 (Level 3)

1.  **神游太虚 (Common)**
    *   **条件**: 摸鱼状态 (IDLE)
    *   **描述**: 闭关中，你的元婴离体而出，在天地间遨游了一圈。
    *   **效果**: 修为 +1000, 心魔 -5
2.  **空间裂隙 (Rare)**
    *   **描述**: 面前的空间突然裂开一道细缝，掉落一块形状扭曲的石头。
    *   **效果**: 获得 [虚空石 (ore_void_stone)] x1
3.  **九曲灵踪 (Rare)**
    *   **条件**: 工作状态 (WORK)
    *   **描述**: 你看见一只白兔钻入土中消失不见，只留下一截红绳。
    *   **效果**: 获得 [养魂草 (herb_soul_restore)] x1
4.  **黄泉回响 (Unique)**
    *   **描述**: 你的元婴误入黄泉河畔，被河水沾湿了衣角，却因祸得福凝练了魂体。
    *   **效果**: 获得 [黄泉水 (water_nether)] x1, 心魔 +10

### 4. 化神期 (Level 4)

1.  **道韵共鸣 (Common)**
    *   **描述**: 你观摩落叶飘零，心中忽然对枯荣法则多了一丝明悟。
    *   **效果**: 修为 +3000
2.  **鲲鹏掠影 (Rare)**
    *   **描述**: 巨大的阴影遮蔽了天空，一根羽毛如山峰般坠落。
    *   **效果**: 获得 [鲲鹏羽 (part_kunpeng_feather)] x1
3.  **息壤再生 (Unique)**
    *   **描述**: 你得到的这捧泥土竟然在吞噬周围的岩石自我增殖！
    *   **效果**: 获得 [息壤 (soil_chaos)] x1

### 5. 炼虚期 (Level 5)

1.  **虚空乱流 (Common)**
    *   **条件**: 工作状态 (WORK)
    *   **描述**: 你在虚空穿行时遭遇了能量乱流，不得不消耗修为抵御。
    *   **效果**: 修为 -5000, 体魄 +5
2.  **死星闪耀 (Rare)**
    *   **描述**: 远处一颗恒星熄灭了，你冒死捕获了它坍缩后的核心。
    *   **效果**: 获得 [星核 (core_star)] x1
3.  **彼岸花开 (Rare)**
    *   **条件**: 心魔 >= 50
    *   **描述**: 在神识恍惚间，你看到了一条满是红花的河流，想起了前世的名字。
    *   **效果**: 获得 [彼岸花 (flower_other_shore)] x1, 心魔 -20

### 6. 合体期 (Level 6)

1.  **万兽朝苍 (Common)**

2.  **麒麟献瑞 (Rare)**
    *   **条件**: 气运 (Luck) >= 10
    *   **描述**: 一头火麒麟踏云而来，吐出一块臂骨后消失在云端。
    *   **效果**: 获得 [麒麟臂 (bone_kirin_arm)] x1
3.  **三生石畔 (Unique)**
    *   **描述**: 你在梦中看到了自己的过去、未来，以及一块刻着名字的石头。
    *   **效果**: 获得 [三生石 (stone_three_life)] x1

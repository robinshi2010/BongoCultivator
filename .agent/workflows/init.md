---
description: 初始化项目文件夹结构
---
1. 创建基础项目目录结构，包括 data, docs, plans, src, tools。
// turbo
2. mkdir -p data docs plans src tools

3. 确保项目根目录下存在 README.md 文件。
// turbo
4. touch README.md

5. 完善 plans 目录结构（包含 active 和 archive 子目录以及说明文件）。
// turbo
6. mkdir -p plans/active plans/archive && touch plans/README.md

7. 检查并确认结构创建完毕。
